info 1. Pendahuluan

SimpleAkunting V4 adalah aplikasi akuntansi berbasis web yang dirancang untuk UMKM, Koperasi, dan BUMDesa di Indonesia. Aplikasi ini mendukung standar akuntansi Indonesia termasuk SAK ETAP dan PSAK 69 untuk aset biologis.

Fitur Utama

üè¢ Multi-Entity üë• Multi-User üìä Laporan Lengkap üåø PSAK 69 üè≠ Manufacturing üì± Responsive üìÑ Export PDF üì• Import Excel

Jenis Entitas yang Didukung

Entitas Keterangan
UMKM Usaha Mikro, Kecil, dan Menengah
Koperasi Koperasi berbagai jenis
BUMDesa Badan Usaha Milik Desa dengan multi unit usaha
Yayasan Organisasi nirlaba

Role Pengguna

Role Hak Akses
Administrator Akses penuh ke semua fitur termasuk manajemen user
Manajer Dapat approve, delete, dan edit semua data
Operator Dapat create dan edit data transaksi
Peninjau Hanya dapat melihat data (read-only)

account_tree 5. Bagan Akun (Chart of Accounts)

Menu: Master Data ‚Üí Bagan Akun

Struktur Akun

Kode Tipe Contoh
1xxx Aset (Asset) Kas, Bank, Piutang, Persediaan
2xxx Kewajiban (Liability) Utang Usaha, Utang Bank
3xxx Ekuitas (Equity) Modal, Laba Ditahan
4xxx Pendapatan (Revenue) Penjualan, Pendapatan Lain
5xxx Beban (Expense) HPP, Beban Operasional

Cara Menambah Akun Baru

  1. Klik Tombol "Tambah Akun" Tombol berwarna primary di pojok kanan
  2. Isi Kode Akun Contoh: 1101 untuk Kas Kecil
  3. Isi Nama Akun Nama deskriptif untuk akun
  4. Pilih Tipe Akun Asset, Liability, Equity, Revenue, atau Expense
  5. Pilih Saldo Normal DEBIT atau CREDIT
  6. Pilih Parent Akun (opsional) Untuk membuat struktur hierarkis
  7. Klik Simpan Akun baru akan muncul di daftar

Import Akun dari Excel

  1. Download Template Klik "Download Template" untuk mendapat format Excel
  2. Isi Template Isi kolom: code, name, type, normal_balance, report_type
  3. Upload File Klik "Import" dan pilih file Excel
  4. Verifikasi Sistem akan memvalidasi dan mengimpor data

domain 8. Aset Tetap (Fixed Assets)

Menu: Master Data ‚Üí Aset Tetap

Informasi Aset Tetap

  • Nama Aset: Deskripsi aset
  • Tanggal Perolehan: Kapan aset dibeli
  • Harga Perolehan: Nilai pembelian
  • Umur Ekonomis: Masa pakai dalam bulan
  • Nilai Residu: Nilai sisa di akhir umur ekonomis
  • Metode Depresiasi: Garis Lurus (Straight Line)

Cara Menambah Aset Tetap

  1. Klik "Tambah Aset"
  2. Isi Data Aset Nama, tanggal perolehan, harga
  3. Isi Depresiasi Umur ekonomis dan nilai residu
  4. Pilih Akun Akun Aset Tetap, Akumulasi Depresiasi, Beban Depresiasi
  5. Simpan Depresiasi akan dihitung otomatis setiap bulan

Rumus Depresiasi Garis Lurus

üìê
Depresiasi per Bulan = (Harga Perolehan - Nilai Residu) / Umur Ekonomis
Contoh: (10.000.000 - 1.000.000) / 60 bulan = Rp 150.000/bulan

edit_note 12. Jurnal

Menu: Transaksi ‚Üí Jurnal

Jenis Jurnal

edit_note Jurnal Umum

Entri manual untuk transaksi non-standar

tune Jurnal Penyesuaian

Koreksi saldo di akhir periode

lock Jurnal Penutup

Menutup akun pendapatan/beban ke laba ditahan

Cara Membuat Jurnal Manual

  1. Buka Jurnal ‚Üí Jurnal Manual
  2. Isi Tanggal
  3. Isi Keterangan Deskripsi transaksi
  4. Tambah Baris Debit Pilih akun dan isi jumlah di kolom Debit
  5. Tambah Baris Credit Pilih akun dan isi jumlah di kolom Credit
  6. Pastikan Balance Total Debit = Total Credit
  7. Simpan
‚ö†Ô∏è
Penting: Total Debit harus sama dengan Total Credit. Jika tidak balance, jurnal tidak bisa disimpan.

precision_manufacturing 14. Manufaktur

14.1 Assembly / Bill of Materials (BOM)

Menu: Manufaktur ‚Üí Assembly

BOM mendefinisikan resep produksi: bahan apa saja yang dibutuhkan untuk menghasilkan produk jadi.

  1. Pilih Produk Jadi Item persediaan kategori "Barang Jadi"
  2. Tambah Komponen Pilih bahan baku dan tentukan kuantitas
  3. Simpan BOM Resep siap digunakan untuk produksi

14.2 Production Order

Menu: Manufaktur ‚Üí Produksi

Status Produksi

draft Draft
play_arrow In Progress
check_circle Completed

Cara Membuat Production Order

  1. Klik "Buat Produksi"
  2. Pilih Assembly/BOM Produk yang akan diproduksi
  3. Isi Quantity Jumlah yang akan diproduksi
  4. Simpan (Draft)
  5. Klik "Start" Status menjadi In Progress, bahan baku dikurangi dari stok
  6. Klik "Complete" Status menjadi Completed, produk jadi masuk stok

analytics 16. Laporan Keuangan

Menu: Laporan

Laporan Utama

Laporan Fungsi Akses
Neraca Posisi keuangan: Aset = Liabilitas + Ekuitas Laporan ‚Üí Neraca
Laba Rugi Kinerja: Pendapatan - Beban = Laba/Rugi Laporan ‚Üí Laba Rugi

Perubahan Ekuitas Mutasi modal/ekuitas Laporan ‚Üí Perubahan Ekuitas

Daftar Jurnal Semua transaksi jurnal Laporan ‚Üí Daftar Jurnal

Cara Melihat Laporan

  1. Pilih Jenis Laporan Dari menu Laporan
  2. Pilih Periode Filter berdasarkan tanggal
  3. Klik Tampilkan Laporan akan ditampilkan di layar
  4. Export PDF (opsional) Klik tombol "Export PDF" untuk download

Laporan Komparatif

Bandingkan laporan 2 periode berbeda:

  1. Buka Laporan (Neraca atau Laba Rugi)
  2. Klik "Komparatif"
  3. Pilih 2 Periode Periode 1 dan Periode 2
  4. Tampilkan Melihat perbandingan side-by-side
